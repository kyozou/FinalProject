## Rock, Paper, Scissors (Final Project)
  A simple GUI program to play Rock, Paper, Scissors against the computer using Python and Tkinter.

## Prerequisites
  Python 3.10 
  Tkinter
  Pillow

## How to Run
  To execute game run the main.py script
